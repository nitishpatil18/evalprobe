# ragmeter

Lightweight, reliable RAG evaluation for solo AI builders. Work in progress.

## What

Evaluate RAG pipelines with 4 core metrics: faithfulness, answer relevancy, context precision, answer correctness. No breaking changes. Works with any LLM via [litellm](https://github.com/BerriAI/litellm).

## Status

Pre-alpha. Building in public. Star to follow.
